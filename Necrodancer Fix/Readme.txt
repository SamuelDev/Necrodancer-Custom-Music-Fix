1. Extract all contents to your custom music folder \Steam\SteamApps\common\Crypt of the NecroDancer\data\custom_music

2. Run beatdown.exe with all of your mp3 files in the same directory, this will generate the .txt files the game needs

3. Run rename.exe to correct the file names for all of the incorrectly named files

4. (Optional) Delete the rename.exe and its related files, or leave them so you can use them again when needed.
